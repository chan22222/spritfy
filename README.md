# Spritfy

Free online pixel art editor, sprite sheet generator, and image converter.

**[https://spritfy.xyz](https://spritfy.xyz)**

- Pixel Art Editor — Multi-layer, animation, onion skin
- Sprite Sheet — Video/GIF frame extraction, chroma key, deduplication
- Image Converter — PNG, JPG, WebP, ICO, GIF, BMP
- Community Gallery — Share and discover pixel art

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
